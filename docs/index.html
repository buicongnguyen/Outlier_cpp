Study Goal

This site is a training environment for preparing for C++ expert assessments and completing Outlier-style coding work. It combines modern C++ knowledge — the memory model, RAII, object-oriented design, architecture, testing, sanitizers, and the standard library — with realistic timed implementation, debugging, code-review, response-ranking, concurrency, and rubric-writing practice. The exercises are original and role-aligned; they are not copied from or guaranteed to match a private assessment.

Language

C++ Language & Memory Model

Value semantics, pointers vs references, const correctness, RAII and smart pointers, move semantics and rvalue references, templates, lambdas, undefined behavior, string_view, and constexpr.

Design

Object-Oriented Design

Virtual dispatch and vtables, abstract interfaces, the rule of zero/three/five, virtual destructors, object slicing, inheritance vs composition, SOLID in C++, CRTP, and PIMPL.

Architecture

Architecture, Toolchain & Ecosystem

Translation units and linking, the ODR, CMake, static vs shared libraries, vcpkg/Conan, GoogleTest, sanitizers, STL containers and algorithms, threads, mutexes, atomics, blocking/non-blocking queues, and error-handling strategy.

Practice format

Question Drills

114 questions in four formats — single answer, select-all-that-apply, click-the-steps-in-order design chains, and fill-in-the-blank code snippets — including a tagged senior/principal tier (production diagnosis, ABI evolution, concurrency tradeoffs). Per-section and overall scores, show-all toggle, reshuffling retry. Works offline.

Eight original tasks, 165-minute full-mock mode, saved drafts, locked reference answers, hidden edge cases, and a 100-point evidence-based rubric. Includes atomics vs mutexes, a closeable MPMC queue, deadlock repair, API lifetime bugs, response evaluation, and implementation.

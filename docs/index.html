Study Goal

This site collects revision material for C++ software-engineering assessment work. The target skill set: strong command of modern C++ including the memory model and RAII, object-oriented design, and application architecture, plus experience with the common C++ toolchain — build systems, testing, sanitizers, and the standard library. Each area pairs a definitions list and core-concept notes with a bank of multiple-choice practice questions and explanations.

Language

C++ Language & Memory Model

Value semantics, pointers vs references, const correctness, RAII and smart pointers, move semantics and rvalue references, templates, lambdas, undefined behavior, string_view, and constexpr.

Design

Object-Oriented Design

Virtual dispatch and vtables, abstract interfaces, the rule of zero/three/five, virtual destructors, object slicing, inheritance vs composition, SOLID in C++, CRTP, and PIMPL.

Architecture

Architecture, Toolchain & Ecosystem

Translation units and linking, the ODR, CMake, static vs shared libraries, vcpkg/Conan, GoogleTest, sanitizers, STL containers and algorithms, std::thread and atomics, and error-handling strategy.

Practice format

Question Drills

90 questions in three formats — single answer, select-all-that-apply, and click-the-steps-in-order design chains — including a tagged senior/principal tier (production diagnosis, ABI evolution, concurrency tradeoffs). Per-section and overall scores, show-all toggle, reshuffling retry. Works offline.
